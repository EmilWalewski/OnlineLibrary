Jackson

One of the tasks you are almost guaranteed to face in every Java project is serializing objects to JSON and deserializing JSON back to Java objects. Jackson is the de facto standard library for this task in the Java ecosystem and it is also the default (de)serializer for Spring MVC and Spring Boot. In most scenarios, Jackson just works under the hood and does the right thing with the default configuration. But sometimes you just need more control over the (de)serialization process.
