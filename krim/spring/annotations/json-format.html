@JsonFormat

It is a Jackson annotation that is used to specify how to format fields and/or properties for JSON output.

Specifically, this annotation allows you to specify how to format Date and Calendar values according to a SimpleDateFormat format.

We can also customize the format of our date fields using the @JsonFormat annotation. If we want our date fields in ISO 8601 format we just need to annotate our field someDate with:

@JsonFormat(shape = JsonFormat.Shape.STRING, pattern= "yyyy-MM-dd")
private Calendar someDate;

@JsonFormat can also be used to serialize subclasses of java.util.Collection and java.lang.Number as POJOs instead of collection elements and Number scalar value respectively. It only works if the annotation is used on class level with @JsonFormat#shape=JsonFormat.Shape.OBJECT

public class ArrayListEx extends AbstractList {
