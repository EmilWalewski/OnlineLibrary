We can annotate any method of the enum with @JsonValue and the result of that method will be the value used for the serialization. Letâ€™s say we annotate the method getA():

OPTION_1("0", "0"), OPTION_2("1", "1");

private String b;

JacksonTestEnum(String a, String b) {

}

@JsonValue

}

}
}
